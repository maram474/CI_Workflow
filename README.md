\# CI/CD Pipeline with Python



Automated tests, code coverage, GitHub Pages deployment, and email notification.



\## Features

\- Pytest tests

\- Coverage reports

\- Artifact collection

\- Deployment to GitHub Pages

\- Email notification with test report



\## Structure

\- `src/` : source code

\- `tests/` : test files

\- `.github/workflows/ci.yml` : CI/CD workflow
